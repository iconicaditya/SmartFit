## Project Description

The SmartFit application is a mobile-based fitness tracking system that allows users to record daily activities such as steps, workouts, and calorie intake. The app also provides fitness and nutrition suggestions through online APIs and helps users monitor their progress with daily and weekly summaries.

---

## Technologies Used

- **Kotlin** - Primary programming language
- **Jetpack Compose** - Modern UI toolkit
- **Room Database** - Local data persistence
- **DataStore** - Secure data storage for preferences
- **Retrofit** - HTTP client for API calls
- **Coroutines** - Asynchronous programming
- **Android Studio** - Development IDE

## Installation / Setup

1. **Install Android Studio**
   - Download from [Android Studio official website](https://developer.android.com/studio)

2. **Clone this repository**
   ```bash
   git clone https://github.com/your-username/smartfit-app.git
   cd SmartFit
   ```

3. **Open the project in Android Studio**
   - File → Open → Select the SmartFit project folder

4. **Sync Gradle files**
   - Android Studio will prompt you to sync Gradle files, or manually go to File → Sync Now

5. **Run the app**
   - On an emulator: Click the green Run button (▶) in the toolbar
   - On a device: Connect your Android device and select it from the device dropdown
